## Running the Presentation

If you are inclined to do so, you can run the presentation in a terminal: 

**Prerequisites:**
- Install [presenterm](https://github.com/mfontanini/presenterm)

**Run the slides:**
```bash
presenterm presentation.md
```

**Tips:**
- Use arrow keys or space to navigate slides
- Press `q` to quit
- The presentation is designed to be shown in split-screen with code on the right
